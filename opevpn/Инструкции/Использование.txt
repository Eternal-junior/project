Вашему вниманию представлен VPN сервер, который является ключевый компонентом исходной инфраструктуры.
Для работы на нем, необходимо залогиниться используя команду:
ssh admin@150.241.114.219 -p 222
Так как ранее был сохранен ключ на вашем компьютер, система не потребует ввод пароля.
При этом при авторизации на сервере, можно сменить пароль, на тот, что наиболее комфортен с помощью команды:
sudo passwd admin
Для того, чтобы создать ключ и реквест на подпись необходимо воспользовать скриптом start.sh:
./start.sh
Система запросит данные по пользователю для создания ключа и реквеста, необходимо следовать инструкции. Чтобы прервать скрипт на определенном этапе, необходимо использовать сочетание клавиш Ctr + C.
По завершению скрипта, программа отправит ключ и сертификат на сервер удостверяющего центра 194.28.224.250, где его можно подписать и выгрузить для отправки пользователю.
Также на сервере действует система мониторинга, в случае возникновения ошибки, можно найти ее решение в файле Инструкции:
cat /Инструкции/"название алерта

Раз в месяц, а именно, 15го числа происходит автоматический backup всех данных необходимых для восстановления, и последующая отправка их на сервер удаленной репозитории 37.46.18.242.
В случаи серьезных технических проблем (вплоть до уничтожения сервера), можно выгрузить самый свежий пакет оттуда и распоковать с помощью
sudo dpkg -i /"рабочая директория/"имя пакета.deb"
Также на сервере действует антивирусная защита rkhanter, поэтому неоходимо проверять логи ошибок по его работе. Данная функция также предусмотрена в автоматических задачах раз в неделю, на ряду с обновлением всех пакетов (apt-get update).
