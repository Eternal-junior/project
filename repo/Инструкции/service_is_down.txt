Данная инструкция предназначена для устранения 3 ошибок, по которым, либо служба nginx, либо node-exporter,либо nginx-exporter недоступны.
1) Проверим статус службы, чтобы исключить возможное ложное срабатывание:
sudo systemctl status "служба".service
Нормальное значение active(running)
2) Если состояние отличается от нормального необходимо произвести рестарт:
sudo systemctl restart "служба".service
и проверить статус.
3.Если служба не запустилась можно проверить содержимое директории с конфиг файлом(касается службы nginx):
sudo nginx -t     Команда должна отработать без ошибок
4) Проверьте ошибки в журнале и сислоге:
sudo journalctl -xe
5) На самый крайний случай, если сервер полностью нужно переустановить можно выгрузить содержимое пакетов с каждого инстанса и установить заново. Потребуется настройка nginx.
